# 프로젝트 정신 & 배경 (SOUL)

## 🎯 프로젝트의 목적

**DSR의 해상풍력 로프 시장 진출을 지원하는 글로벌 시장 모니터링 시스템**

- **회사**: DSR (합성섬유로프 제조, 27년 경력)
- **목표**: 해상풍력 로프 시장 진출
- **방법**: 온디맨드 시장 정보 수집 및 분석

## 📖 프로젝트 역사

### Phase 1: 초기 구축 (2026-06-11)
- **문제**: 해상풍력 로프 시장 정보 필요하지만 너무 방대함
- **해결**: 자동화 시스템 구축
- **결과**: 매주 자동 리포트 (하지만 Token 낭비, 시간 오래 걸림)

### Phase 2: 최적화 (2026-06-11)
- **문제**: Token 사용 너무 많음, 시간 오래 걸림, 논문 필요 없음
- **분석**: 
  - 논문 조사는 불필요 (산업 뉴스로 충분)
  - 전 세계 자료는 너무 광범위
  - 자동화는 비효율 (필요할 때만)
  
- **개선**:
  - ✅ 2025년 자료만 조사
  - ✅ 유럽, 중국, 한국만 조사
  - ✅ 논문 제외 (산업 뉴스/규제만)
  - ✅ 온디맨드 방식 (필요할 때만)
  - ✅ Token 60% 절감
  - ✅ 시간 75% 단축

## 💡 핵심 가치관

### 1. **실용성 우선**
- 이론보다 현장 정보
- 학문보다 산업 동향
- 자동화보다 효율성

### 2. **필요한 것만**
- 세계 모든 정보 → 3개 지역만
- 과거~현재 → 2025년만
- 학술지 포함 → 뉴스/규제만

### 3. **효율 극대화**
- Token 절감 (60% 감소)
- 시간 단축 (75% 단축)
- 자동화 제거 (온디맨드로)

### 4. **의사결정 지원**
- 정보는 결정을 위함
- 분석은 행동을 위함
- 리포트는 실행을 위함

## 🔍 조사 범위 결정

### 지역: 왜 유럽, 중국, 한국?

**유럽 🇪🇺**
- 해상풍력 기술 선진지
- 국제 표준 주도 (DNV, ABS)
- DSR의 글로벌 진출 모델

**중국 🇨🇳**
- 2025년 가장 빠른 성장 (1,400만 kW)
- 가장 큰 시장
- 기술 추격 압박

**한국 🇰🇷**
- DSR의 홈마켓
- 안마(532MW), 해송(1GW) 프로젝트
- 국산화 강화 추세

**제외**: 미국, 일본, 동남아 등
- 이유: 정보 과다, 우선순위 낮음
- 필요 시 별도 조사

### 기간: 왜 2025년만?

**최신 트렌드 파악**
- 5년 이상 과거 데이터는 변화 추격 불가
- 2025년 최신 정보가 의사결정에 직접 영향
- 과거 데이터는 필요시 별도 조사

**예:**
```
2020-2024: 배경 지식 (필요시)
2025: 현재 결정 (항상)
```

### 자료: 왜 논문 제외?

**논문의 문제점**
- ❌ 발행 2-3년 후 (너무 늦음)
- ❌ 학술적 (실무와 괴리)
- ❌ 조사 시간 과다 (Token 낭비)

**산업 뉴스/규제의 장점**
- ✅ 실시간 (며칠 내)
- ✅ 실무 중심 (즉시 적용)
- ✅ 빠른 조사 (효율적)

**예:**
```
논문: "2023년 합성 로프 기술 분석" (2년 후 발행)
뉴스: "Lankhorst 2025년 신제품 출시" (실시간)
→ 뉴스가 의사결정에 더 유용
```
