# Claude Code 협업 지시사항

DSR 해상풍력 로프 시장 프로젝트에서 Claude와 협업하기 위한 규칙입니다.

## 🎯 협업 원칙

- **언어**: 한국어 선호
- **스타일**: 간단명료 (복잡한 과정 없이)
- **속도**: 빠른 실행 우선
- **품질**: 최소 필요 수준 유지
- **효율**: Token & 시간 최적화

## 📝 작업 요청 방식

### 명확한 요청
```
"해상풍력 로프 시장 리포트 만들어줘 (2025년, 유럽/중국/한국)"
```

### 불명확한 요청
```
❌ "개선점 찾아줘" → ✅ "README 오류 찾아줘"
❌ "코드 짜줘" → ✅ "Python으로 메모리 초기화 스크립트 작성"
```

## 🔍 검색 및 조사

**온디맨드 리포트 (자동 검증 포함):**
- 2025년 자료만
- 유럽, 중국, 한국만
- 논문 제외 (산업 뉴스/규제만)
- Token 효율적 (병렬 검색)
- **검증 스킬 자동 실행** (중복 방지, 범위 확인, 품질 보증)

**프로세스:**
```
사용자 요청
  ↓
검증 스킬 (offshore-rope-research-validator)
  ├─ 중복 확인
  ├─ 범위 확인
  └─ 진행/수정 제안
  ↓
리포트 생성 (또는 기존 리포트 제시)
  ↓
검증 스킬 (생성 후 품질 확인)
  ├─ 구조 확인
  ├─ 범위 준수 확인
  ├─ 출처 신뢰성 확인
  └─ 개선사항 제안
```

**예시:**
```
사용자: "해상풍력 로프 시장 리포트 생성"
→ 검증 스킬: ✅ 범위 명확, 중복 없음
→ 리포트 생성: 약 2분 내 완성 (~42K Token)
→ 검증 스킬: ✅ 품질 보증, 개선사항 제안
```
